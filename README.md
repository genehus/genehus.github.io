# GeneHus

AI-powered genomic diagnostics for earlier cancer detection across Africa.

## About

GeneHus combines artificial intelligence and genomics to identify biomarkers and stratify cancer risk earlier and more accurately—empowering clinicians and researchers across Africa.

## Site

This repository hosts the GeneHus marketing website, deployed via [GitHub Pages](https://genehus.github.io/).

## Structure

- `index.html` — page markup
- `style.css` — styles
- `script.js` — navigation and scroll behavior
- `assets/` — images, logos, and section artwork (`man_1.png`–`man_12.png`)

## Local development

Open `index.html` in a browser, or serve the folder with any static file server:

```bash
npx serve .
```
